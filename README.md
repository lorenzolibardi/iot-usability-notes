# IoT Usability Notes

This repository contains lecture notes and teaching material on **usability and human-centered design in Internet of Things (IoT) systems**.
All the notes and materials in this repository are written in **Italian**.

## Contents

The material covers topics such as:
- Usability challenges in IoT systems;
- Human-centered design principles applied to IoT;
- Constraints of IoT devices, including limited interfaces and context of use;
- Design guidelines and real-world examples.

## Intended Audience

These notes are intended for:
- University students;
- Instructors and teaching assistants;
- Researchers and practitioners interested in usability and IoT design.

## Note

This material is shared for educational purposes.
